## Features

- Next.js 14
- React 18
- TypeScript 5
- ESLint & Prettier
- Husky & lint-staged
- CI/CD workflow
- Renovate

## Getting Started

1. install dependencies

```
npm install
```

2. run server

```
// dev
npm run dev

// prod
npm run start
```
